# ZabdyCart 🛒 Responsive Eccomerce Website

A modern, fast, and fully responsive e-commerce frontend interface built with sleek typography, real-time interactivity, and a clean corporate layout.

---

## 🚀 Key Features

* **Dynamic Tab Views:** Seamlessly switches between products and dedicated info screens. Clicking "About Us" or "Contact" automatically hides categories and displays structured content instantly without reloading the page.
* **Live Shopping Cart:** Features real-time item tracking with dynamic price aggregation and badge update functionality.
* **Smart Wishlist Engine:** Interactive wishlist toggles with live color-changing state highlights.
* **Responsive Navigation:** Clean multi-device category dropdown menu designed for standard modern displays.
* **Premium Theme UI:** High-end aesthetic color grading using a custom corporate palette of Desaturated Teal (`#0E6464`) and Coral (`#F15E4A`).

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Flexbox, CSS Grid, Custom Root Variables)
* **Interactivity:** Pure Vanilla JavaScript (DOM Manipulation & State Management)
* **Typography & Icons:** Google Fonts (Poppins), FontAwesome v6.4.0
* **Data Layer:** Normalized relational database structure ready for backend API pipeline integrations.

---

## 💻 How to Run the Project Locally

Follow these clear, step-by-step instructions to preview the project manually on your machine:

### 1. Project Directory Setup
Ensure that all core files (`index.html`, `style.css`, and `script.js`) are saved together in the same root folder.

### 2. Launch Local Static Server
You can run this project using a standard light-weight server. If you have **Node.js** installed, execute the following commands in your terminal inside the project directory:

```bash
# Install the global live-server dependency utility
npm install -g live-server

# Initialize and start your local development server
live-server
3. Open the Application URL
Once the terminal successfully launches the server, copy and paste the local URL output directly into your web browser:

Plaintext
http://localhost:8080
(Note: Use the specific active port number generated by your command prompt environment).
